## Bash port of [i3-swallow](https://github.com/jamesofarrell/i3-swallow)

## Requirements
* bash
* i3-msg
* jq

## Installation
Make it executable
```sh
chmod u+x swallow
```

## Usage
```sh
./swallow -h
```
